# C* Music Player (cmus) Logo

I needed a logo for C* Music Player ([cmus](https://github.com/cmus/)) and since there isn't one, I fired up [Inkscape](https://inkscape.org/) and quickly created one. The cmus logo, like cmus itself, is released under the [GPL 2.0 license](https://www.gnu.org/licenses/old-licenses/gpl-2.0.en.html). Anyone can use the logo for free.

Cheers, [J15k](https://www.j15k.com/ "Jonas Jared Jacek").


## cmus Vector Logos

The SVG (Scalable Vector Graphics) logo uses a transparent background.
